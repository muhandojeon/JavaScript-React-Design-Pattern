# 디자인 패턴의 유형

디자인 패턴은 어떤 문제를 해결하느냐에 따라 다음 세 가지 유형으로 분류된다.

- 생성 패턴
- 구조 패턴
- 행위 패턴

## 6.2 생성 패턴

기본적인 디자인 패턴을 사용하지 않은 객체 생성 방식은 프로젝트의 복잡성을 증가시킬 수도 있기에 *이 과정을 제어하여 문제를 해결*한다.

- 생성자 패턴
- 팩토리 패턴
- 추상 팩토리 패턴
- 싱글톤 패턴
- 프로토타입 패턴
- 빌더 패턴

## 6.3 구조 패턴

객체의 구성과 각 객체 간의 관계를 인식하는 방법에 중점을 둡니다.

- 데코레이터 패턴
- 퍼사드 패턴
- 플라이웨이트 패턴
- 어댑터 패턴
- 프록시 패턴

## 6.4 행위 패턴

객체 간의 책임 분배와 협력 방법에 중점을 둡니다.

- 이터레이터 패턴
- 중재자 패턴
- 옵저버 패턴
- 방문자 패턴
